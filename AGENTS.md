# Design

All UI work follows `DESIGN.md` (the Floqex design system). Read it before
building or changing any interface, and keep every surface consistent with it.
Before any design work, always invoke the taste skill (`npx skills add
Leonxlnx/taste-skill`, `npx impeccable install`, and load `design-taste-frontend`)
and run its anti-slop pre-flight. The product is dark-locked, single emerald
accent, soft-rounded, real-data-only, with zero em-dashes.
